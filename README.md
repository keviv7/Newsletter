# Newsletter

## Dependencies:

* python>=3.4
* pipenv 
* nodejs >= 12.13.0 
* yarn

## Usage:


```
git clone https://github.com/keviv7/Newsletter.git
cd Newsletter/backend
pipenv shell
pipenv install --skip-lock
python manage.py runserver
```

Now, open a new Terminal window:

```
cd Newsletter/frontend
yarn install
yarn start
```
